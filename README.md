## Description

Coding exercises with their unit tests

## Getting Started

### Dependencies

* Unix based OS
* Python 3

### Installing

* git clone git@github.com:dmb225/algorithms.git
* cd algorithms
* virtualenv --python=python3.8 venv --no-site-packages
* source venv /bin/activate
* pip install -r requirements.txt

### Executing program

* pytest source
* flake8 source
